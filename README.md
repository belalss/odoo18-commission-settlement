# Odoo 18 Commission Settlement Wizard

Lightweight custom addon for Odoo 18 Community.

Features:
- Generate commission settlements from paid invoices
- Uses OCA commission lines
- Period based settlement
- Agent + invoice breakdown

Built for demo and MVP use.

Requirements:
- Odoo 18 Community
- OCA: commission_oca, sale_commission_oca, account_commission_oca
